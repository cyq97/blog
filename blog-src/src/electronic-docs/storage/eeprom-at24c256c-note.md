# EEPROM AT24C256C 笔记



**设备地址**

> 回顾：I2C 通信时，作为 8-bit 地址的最低位为读/写标志(0:读，1:写)

可通过引脚 A2-A1-A0 配置 I2C 通信时的设备地址

| 7-bit Slave Address |
| :-----------------: |
|  1010_A2-A1-A0-R/W  |



**存储器地址**

数据访问地址宽度：15-bit

实际 I2C 通信时，使用两字节表示数据访问地址，**地址的高字节先传输**。



**存储器组织结构**

- Page Size: 64-byte

- Page Count: 512

- (512 * 64)-byte =  (256 * 1024)-bit
- Memory address range: 0x0000 ~ (0x8000-1)
