# OLED SSH1106 笔记





## I2C 通信

**通信速率**

MAX: 400KHz

<br>

**设备地址**

> 回顾：I2C 通信时，作为 8-bit 地址的最低位为读/写标志(0:读，1:写)

可通过引脚 SA0 配置 I2C 通信时的设备地址

| SA0  | 7-bit Slave Address | 8-bit Write(0) | 8-bit Read(1) |
| :--: | :-----------------: | :------------: | :-----------: |
|  0   |    0111_100_R/W     |      0x78      |     0x79      |
|  1   |    0111_101_R/W     |      0x7A      |     0x7B      |
